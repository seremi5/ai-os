# Profile — identity & voice

> L1 Context · Identity. Apply this without being asked. ≤500 words.

## Who I am

Sergi Reina-Miret. AI Product Manager at **OakNorth** (UK digital bank, SME lending). A lawyer who became a fintech PM — ex-**Monite** (embedded finance / AP-AR), ex-**Vertex** and **Avalara** (tax / VAT / e-invoicing). I build AI tools as a serious side practice: evenings and weekends, not external funding.

I'm technical, design-literate, and an experienced builder. My capital is my time. A plausible side project is one a single competent person can prototype to a credible v1 in a week and monetise within 90 days.

## How I write

- **British English. Plain words.** Explain as if to a smart colleague whose first language is Spanish.
- **Pyramid Principle.** Lead with the answer. Support it with MECE arguments. Never build towards a conclusion.
- **Concrete over abstract.** A real number, a named tool, a specific example beats a category noun.
- **Short.** If a paragraph can be a sentence, make it a sentence. If a file passes 500 words, split it.
- Banned vocabulary lives in [`terminology.md`](terminology.md). Read it before writing anything public-facing.

## How I work

- **Ship small.** Smallest change that proves the point, then iterate. No big-bang rewrites.
- **Direct provider APIs over SaaS wrappers.** I'd rather call Anthropic/Gemini directly than rent a middle layer.
- **Regression-tested incremental change.** Don't break what works to add what's new.
- **Filesystem as durable memory.** Artefacts are plain markdown in folders, not chat. The work survives a restart.
- **Agents do one thing.** Single responsibility, single entry point. (See [`engineering/`](../engineering/).)
- **Scoring is not AI.** Quantitative signals are computed deterministically before AI sees them.

## Principles (the one-way doors)

1. **The operator is in the loop.** Nothing ships, nothing is spent, without my approval.
2. **Output volume is not progress.** Judge a cycle against one outcome metric, not page count.
3. **Adversarial by design.** I want the strongest case against, not a polite nod. Don't soften critiques.
4. **Approve every memory.** No silent accumulation. Write the WHY or don't save it.

## What I'm not

Not a fundraiser chasing scale. Not a prompt-tinkerer who never ships. Not interested in generic, voice-less output — if it could've been written for anyone, it's wrong.
