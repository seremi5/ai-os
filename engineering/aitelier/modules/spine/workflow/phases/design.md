# Phase · Design

**Intent:** settle the UX before architecture. Wireframes, user journeys, component reuse.

**Owner:** `designer` (needs `agents:designer`; pairs with `design-system` if installed).

**Entry:** user-facing UI work.

**Do:**
- Write a design PRD in `docs/design/` (use `prd-template.md`).
- Produce wireframes in Figma — a URL, not ASCII art.
- Identify design-system reuse vs. new components; document one journey per section.
- List open UX questions before handing to Architect.

**Gate (exit):** design PRD + Figma link saved; reuse-vs-custom decided; open questions listed.
